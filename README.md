# ITP-unconf-node-workshop
Live, totally improvised code from an [ITP Unconference](http://itp.nyu.edu/unconference) workshop about node.js.

Topics:
- Node.js
- express
- database (mongodb via mongoose)
- creating an api
- templating engines (hogan / mustache)
- heroku
